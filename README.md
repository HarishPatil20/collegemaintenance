# College Maintenance / Campus Management System

A web-based College Maintenance / Campus Management System designed to manage students, faculty, departments, subjects, exams, results, timetables, circulars, and other academic activities in one centralized platform. This project helps colleges automate and simplify daily administrative and academic work.

## âš™ï¸ How to Run Locally

1. Clone the repository: git clone https://github.com/HarishPatil20/collegemaintenance.git

2. Create a database in MySQL (e.g., `collegemaintenance`) and import the SQL file.

3. Update database credentials in your config/connection file.

4. Place the project in:
- `htdocs` (XAMPP) or  
- `webapps` (Tomcat)

5. Start Apache/Tomcat & MySQL and open: http://localhost/collegemaintenance
