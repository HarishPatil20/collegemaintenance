# College Maintenance / Campus Management System

A web-based College Maintenance / Campus Management System designed to manage students, faculty, departments, courses, exams, results, timetables, circulars, and other academic activities in a centralized platform.

This project helps colleges automate and simplify day-to-day administrative and academic operations.
